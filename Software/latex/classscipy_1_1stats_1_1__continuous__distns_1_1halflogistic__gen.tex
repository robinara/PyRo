\subsection{Detailed Description}
\begin{DoxyVerb}A half-logistic continuous random variable.

%(before_notes)s

Notes
-----
The probability density function for `halflogistic` is::

    halflogistic.pdf(x) = 2 * exp(-x) / (1+exp(-x))**2 = 1/2 * sech(x/2)**2

for ``x >= 0``.
