\subsection{Detailed Description}
Internal element class. 

This class defines the Element interface, and provides a reference implementation of this interface. 

You should not create instances of this class directly. Use the appropriate factory functions instead, such as \hyperlink{namespacePyQt-x11-gpl-4_811_82_1_1elementtree_1_1ElementTree_a0144f68b4594417eaaf4ed282f9106db}{Element} and \hyperlink{namespacePyQt-x11-gpl-4_811_82_1_1elementtree_1_1ElementTree_a782e0576bbdc231d6ca3b0bbaf908ea8}{Sub\+Element}.
