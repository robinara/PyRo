{\bfseries  S\+S\+T\+E\+V computes the eigenvalues and, optionally, the left and/or right eigenvectors for O\+T\+H\+E\+R matrices} 

 \begin{DoxyParagraph}{Purpose\+: }
\begin{DoxyVerb} SSTEV computes all eigenvalues and, optionally, eigenvectors of a
 real symmetric tridiagonal matrix A.\end{DoxyVerb}
 
\end{DoxyParagraph}
