{\bfseries D\+L\+A\+E\+I\+N} computes a specified right or left eigenvector of an upper Hessenberg matrix by inverse iteration. 

 \begin{DoxyParagraph}{Purpose\+: }
\begin{DoxyVerb} DLAEIN uses inverse iteration to find a right or left eigenvector
 corresponding to the eigenvalue (WR,WI) of a real upper Hessenberg
 matrix H.\end{DoxyVerb}
