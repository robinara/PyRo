{\bfseries C\+H\+E\+T\+R\+S} 

 \begin{DoxyParagraph}{Purpose\+: }
\begin{DoxyVerb} CHETRS solves a system of linear equations A*X = B with a complex
 Hermitian matrix A using the factorization A = U*D*U**H or
 A = L*D*L**H computed by CHETRF.\end{DoxyVerb}
 
\end{DoxyParagraph}
