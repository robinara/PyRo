{\bfseries Z\+L\+A\+H\+Q\+R} computes the eigenvalues and Schur factorization of an upper Hessenberg matrix, using the double-\/shift/single-\/shift Q\+R algorithm. 

 \begin{DoxyParagraph}{Purpose\+: }
\begin{DoxyVerb}    ZLAHQR is an auxiliary routine called by CHSEQR to update the
    eigenvalues and Schur decomposition already computed by CHSEQR, by
    dealing with the Hessenberg submatrix in rows and columns ILO to
    IHI.\end{DoxyVerb}
 
\end{DoxyParagraph}
