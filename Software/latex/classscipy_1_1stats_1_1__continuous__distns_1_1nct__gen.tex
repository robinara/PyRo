\subsection{Detailed Description}
\begin{DoxyVerb}A non-central Student's T continuous random variable.

%(before_notes)s

Notes
-----
The probability density function for `nct` is::

                                        df**(df/2) * gamma(df+1)
    nct.pdf(x, df, nc) = ----------------------------------------------------
                         2**df*exp(nc**2/2) * (df+x**2)**(df/2) * gamma(df/2)

for ``df > 0``.
