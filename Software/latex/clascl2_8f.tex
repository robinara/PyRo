{\bfseries C\+L\+A\+S\+C\+L2} performs diagonal scaling on a vector. 

 \begin{DoxyParagraph}{Purpose\+: }
\begin{DoxyVerb} CLASCL2 performs a diagonal scaling on a vector:
   x <-- D * x
 where the diagonal REAL matrix D is stored as a vector.

 Eventually to be replaced by BLAS_cge_diag_scale in the new BLAS
 standard.\end{DoxyVerb}
 
\end{DoxyParagraph}
