{\bfseries C\+L\+A\+R\+F\+G\+P} generates an elementary reflector (Householder matrix) with non-\/negatibe beta. 

 \begin{DoxyParagraph}{Purpose\+: }
\begin{DoxyVerb} CLARFGP generates a complex elementary reflector H of order n, such
 that

       H**H * ( alpha ) = ( beta ),   H**H * H = I.
              (   x   )   (   0  )

 where alpha and beta are scalars, beta is real and non-negative, and
 x is an (n-1)-element complex vector.  H is represented in the form

       H = I - tau * ( 1 ) * ( 1 v**H ) ,
                     ( v )

 where tau is a complex scalar and v is a complex (n-1)-element
 vector. Note that H is not hermitian.

 If the elements of x are all zero and alpha is real, then tau = 0
 and H is taken to be the unit matrix.\end{DoxyVerb}
 
\end{DoxyParagraph}
