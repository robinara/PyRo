\subsection{Detailed Description}
\begin{DoxyVerb}A fatigue-life (Birnbaum-Sanders) continuous random variable.

%(before_notes)s

Notes
-----
The probability density function for `fatiguelife` is::

    fatiguelife.pdf(x, c) =
        (x+1) / (2*c*sqrt(2*pi*x**3)) * exp(-(x-1)**2/(2*x*c**2))

for ``x > 0``.
