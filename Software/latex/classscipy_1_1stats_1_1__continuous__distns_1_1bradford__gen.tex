\subsection{Detailed Description}
\begin{DoxyVerb}A Bradford continuous random variable.

%(before_notes)s

Notes
-----
The probability density function for `bradford` is::

    bradford.pdf(x, c) = c / (k * (1+c*x)),

for ``0 < x < 1``, ``c > 0`` and ``k = log(1+c)``.
