{\bfseries  C\+H\+E\+E\+V\+D computes the eigenvalues and, optionally, the left and/or right eigenvectors for H\+E matrices} 

 \begin{DoxyParagraph}{Purpose\+: }
\begin{DoxyVerb} CHEEVD computes all eigenvalues and, optionally, eigenvectors of a
 complex Hermitian matrix A.  If eigenvectors are desired, it uses a
 divide and conquer algorithm.

 The divide and conquer algorithm makes very mild assumptions about
 floating point arithmetic. It will work on machines with a guard
 digit in add/subtract, or on those binary machines without guard
 digits which subtract like the Cray X-MP, Cray Y-MP, Cray C-90, or
 Cray-2. It could conceivably fail on hexadecimal or decimal machines
 without guard digits, but we know of none.\end{DoxyVerb}
 
\end{DoxyParagraph}
