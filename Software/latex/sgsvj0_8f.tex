{\bfseries S\+G\+S\+V\+J0} pre-\/processor for the routine sgesvj. 

 \begin{DoxyParagraph}{Purpose\+: }
\begin{DoxyVerb} SGSVJ0 is called from SGESVJ as a pre-processor and that is its main
 purpose. It applies Jacobi rotations in the same way as SGESVJ does, but
 it does not check convergence (stopping criterion). Few tuning
 parameters (marked by [TP]) are available for the implementer.\end{DoxyVerb}
 
\end{DoxyParagraph}
