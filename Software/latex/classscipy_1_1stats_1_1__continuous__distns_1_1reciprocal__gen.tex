\subsection{Detailed Description}
\begin{DoxyVerb}A reciprocal continuous random variable.

%(before_notes)s

Notes
-----
The probability density function for `reciprocal` is::

    reciprocal.pdf(x, a, b) = 1 / (x*log(b/a))

for ``a <= x <= b``, ``a, b > 0``.
