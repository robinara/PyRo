{\bfseries S\+L\+A\+S\+R\+T} sorts numbers in increasing or decreasing order. 

 \begin{DoxyParagraph}{Purpose\+: }
\begin{DoxyVerb} Sort the numbers in D in increasing order (if ID = 'I') or
 in decreasing order (if ID = 'D' ).

 Use Quick Sort, reverting to Insertion sort on arrays of
 size <= 20. Dimension of STACK limits N to about 2**32.\end{DoxyVerb}
 
\end{DoxyParagraph}
