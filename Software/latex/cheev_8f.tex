{\bfseries  C\+H\+E\+E\+V computes the eigenvalues and, optionally, the left and/or right eigenvectors for H\+E matrices} 

 \begin{DoxyParagraph}{Purpose\+: }
\begin{DoxyVerb} CHEEV computes all eigenvalues and, optionally, eigenvectors of a
 complex Hermitian matrix A.\end{DoxyVerb}
 
\end{DoxyParagraph}
