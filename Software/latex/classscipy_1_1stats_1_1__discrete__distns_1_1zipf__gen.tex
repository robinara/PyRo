\subsection{Detailed Description}
\begin{DoxyVerb}A Zipf discrete random variable.

%(before_notes)s

Notes
-----
The probability mass function for `zipf` is::

    zipf.pmf(k, a) = 1/(zeta(a) * k**a)

for ``k >= 1``.

`zipf` takes ``a`` as shape parameter.
