\subsection{Detailed Description}
\begin{DoxyVerb}An exponentiated Weibull continuous random variable.

%(before_notes)s

Notes
-----
The probability density function for `exponweib` is::

    exponweib.pdf(x, a, c) =
        a * c * (1-exp(-x**c))**(a-1) * exp(-x**c)*x**(c-1)

for ``x > 0``, ``a > 0``, ``c > 0``.
