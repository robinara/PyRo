115 /*
116  * tnc : minimize a function with variables subject to bounds, using
117  * gradient information.
118  *
119  * n : number of variables (must be >= 0)
120  * x : on input, initial estimate ; on output, the solution
121  * f : on output, the function value at the solution
122  * g : on output, the gradient value at the solution
123  * g should be an allocated vector of size n or NULL,
124  * in which case the gradient value is not returned.
125  * function : the function to minimize (see tnc_function)
126  * state : used by function (see tnc_function)
127  * low, up : the bounds
128  * set low[i] to -HUGE_VAL to remove the lower bound
129  * set up[i] to HUGE_VAL to remove the upper bound
130  * if low == NULL, the lower bounds are removed.
131  * if up == NULL, the upper bounds are removed.
132  * scale : scaling factors to apply to each variable
133  * if NULL, the factors are up-low for interval bounded variables
134  * and 1+|x] for the others.
135  * offset : constant to substract to each variable
136  * if NULL, the constant are (up+low)/2 for interval bounded
137  * variables and x for the others.
138  * messages : see the tnc_message enum
139  * maxCGit : max. number of hessian*vector evaluation per main iteration
140  * if maxCGit == 0, the direction chosen is -gradient
141  * if maxCGit < 0, maxCGit is set to max(1,min(50,n/2))
142  * maxnfeval : max. number of function evaluation
143  * eta : severity of the line search. if < 0 or > 1, set to 0.25
144  * stepmx : maximum step for the line search. may be increased during call
145  * if too small, will be set to 10.0
146  * accuracy : relative precision for finite difference calculations
147  * if <= machine_precision, set to sqrt(machine_precision)
148  * fmin : minimum function value estimate
149  * ftol : precision goal for the value of f in the stoping criterion
150  * if ftol < 0.0, ftol is set to accuracy
151  * xtol : precision goal for the value of x in the stopping criterion
152  * (after applying x scaling factors)
153  * if xtol < 0.0, xtol is set to sqrt(machine_precision)
154  * pgtol : precision goal for the value of the projected gradient in the
155  * stopping criterion (after applying x scaling factors)
156  * if pgtol < 0.0, pgtol is set to 1e-2 * sqrt(accuracy)
157  * setting it to 0.0 is not recommended
158  * rescale : f scaling factor (in log10) used to trigger f value rescaling
159  * if 0, rescale at each iteration
160  * if a big value, never rescale
161  * if < 0, rescale is set to 1.3
162  * nfeval : on output, the number of function evaluations.
163  * ignored if nfeval==NULL.
164  *
165  * The tnc function returns a code defined in the tnc_rc enum.
166  * On output, x, f and g may be very slightly out of sync because of scaling.
167  *
168  */
