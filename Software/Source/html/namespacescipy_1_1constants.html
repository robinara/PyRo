.. currentmodule:: scipy.constants

``hbar``       :math:`\hbar = h/(2\pi)`

``m_p``        proton mass
``m_n``        neutron mass
=============  =================================================================


Constants database
------------------

In addition to the above variables, :mod:`scipy.constants` also contains the
2010 CODATA recommended values [CODATA2010]_ database containing more physical
constants.
