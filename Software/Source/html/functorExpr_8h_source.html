20 /* This header file is designed to allow the use of Blitz++ with
21  functors (classes defining an operator()) and more general member
22  functions. It works best if you have access to the class source code;
23  there is limited support for classes that cannot be modified. The best
24  approach in that case is usually to write an adapter class.
25 
26  This works with class methods that take one, two or three arguments.
27 
28  If you have a functor, add the following to your (public) class declaration:
29 
30  BZ_DECLARE_FUNCTOR(classname) // for one argument functors
31  BZ_DECLARE_FUNCTOR2(classname) // for two argument functors
32  BZ_DECLARE_FUNCTOR3(classname) // for three argument functors
33 
34  or
35 
36  BZ_DECLARE_FUNCTOR_RET(classname, returnType)
37  BZ_DECLARE_FUNCTOR2_RET(classname, returnType)
38  BZ_DECLARE_FUNCTOR3_RET(classname, returnType)
39 
40  for classes whose operator() has a return type that is not what you would
41  deduce from the usual C++ promotion rules (e.g., takes two doubles and
42  returns a bool).
43 
44  You can then use your class in Blitz++ expressions and no temporaries will
45  be generated. For example, assuming that your class is named T, and that
46  A, B and C are Arrays, you can write
47 
48  T classInstance( ... );
49  A = C + classInstance(B * tensor::i);
50  A = C + classInstance(tensor::i, tensor::j)
51 
52  It also works for member functions:
53 
54  BZ_DECLARE_MEMBER_FUNCTION(classname, funcname)
55  BZ_DECLARE_MEMBER_FUNCTION2(classname, funcname)
56  BZ_DECLARE_MEMBER_FUNCTION3(classname, funcname)
57 
58  or
59 
60  BZ_DECLARE_MEMBER_FUNCTION_RET(classname, funcname, returnType)
61  BZ_DECLARE_MEMBER_FUNCTION2_RET(classname, funcname, returnType)
62  BZ_DECLARE_MEMBER_FUNCTION3_RET(classname, funcname, returnType)
63 
64  allows you to write stuff like
65 
66  A = C + classInstance.funcname(B * tensor::i);
67  A = C + classInstance.funcname(tensor::i, tensor::j)
68 
69  All the member functions to be applied must be declared const.
70 
71  There is also some support for classes where the source code is not
72  available or not to be tampered with. For example,
73 
74  A = C + applyFunctor(classInstance, B * tensor::i);
75  A = C + applyFunctor(classInstance, tensor::i, tensor::j);
76 
77  This approach does not work for arbitrary member functions. The
78  class must be a proper functor with an operator().
79 
80 */
