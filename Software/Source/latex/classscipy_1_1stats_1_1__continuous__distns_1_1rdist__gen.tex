\subsection{Detailed Description}
\begin{DoxyVerb}An R-distributed continuous random variable.

%(before_notes)s

Notes
-----
The probability density function for `rdist` is::

    rdist.pdf(x, c) = (1-x**2)**(c/2-1) / B(1/2, c/2)

for ``-1 <= x <= 1``, ``c > 0``.
