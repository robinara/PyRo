See Also
--------
johnsonsb

Notes
-----
The probability density function for `johnsonsu` is::

    johnsonsu.pdf(x, a, b) = b / sqrt(x**2 + 1) *
                             phi(a + b * log(x + sqrt(x**2 + 1)))

for all ``x, a, b > 0``, and `phi` is the normal pdf.
