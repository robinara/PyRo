\subsection{Detailed Description}
\begin{DoxyVerb}A beta continuous random variable.

%(before_notes)s

Notes
-----
The probability density function for `beta` is::

    beta.pdf(x, a, b) = gamma(a+b)/(gamma(a)*gamma(b)) * x**(a-1) *
    (1-x)**(b-1),

for ``0 < x < 1``, ``a > 0``, ``b > 0``.
