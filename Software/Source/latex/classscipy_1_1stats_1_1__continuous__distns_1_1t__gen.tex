\subsection{Detailed Description}
\begin{DoxyVerb}A Student's T continuous random variable.

%(before_notes)s

Notes
-----
The probability density function for `t` is::

                                   gamma((df+1)/2)
    t.pdf(x, df) = ---------------------------------------------------
                   sqrt(pi*df) * gamma(df/2) * (1+x**2/df)**((df+1)/2)

for ``df > 0``.
