{\bfseries Z\+H\+E\+T\+R\+S2} 

 \begin{DoxyParagraph}{Purpose\+: }
\begin{DoxyVerb} ZHETRS2 solves a system of linear equations A*X = B with a complex
 Hermitian matrix A using the factorization A = U*D*U**H or
 A = L*D*L**H computed by ZHETRF and converted by ZSYCONV.\end{DoxyVerb}
 
\end{DoxyParagraph}

          = 'L':  Lower triangular, form is A = L*D*L**H.\end{DoxyVerb}
