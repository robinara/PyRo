{\bfseries Z\+H\+P\+T\+R\+S} 

 \begin{DoxyParagraph}{Purpose\+: }
\begin{DoxyVerb} ZHPTRS solves a system of linear equations A*X = B with a complex
 Hermitian matrix A stored in packed format using the factorization
 A = U*D*U**H or A = L*D*L**H computed by ZHPTRF.\end{DoxyVerb}
 
\end{DoxyParagraph}

          = 'U':  Upper triangular, form is A = U*D*U**H;
          = 'L':  Lower triangular, form is A = L*D*L**H.\end{DoxyVerb}
\\
\hline
