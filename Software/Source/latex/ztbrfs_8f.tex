{\bfseries Z\+T\+B\+R\+F\+S} 

 \begin{DoxyParagraph}{Purpose\+: }
\begin{DoxyVerb} ZTBRFS provides error bounds and backward error estimates for the
 solution to a system of linear equations with a triangular band
 coefficient matrix.

 The solution matrix X must be computed by ZTBTRS or some other
 means before entering this routine.  ZTBRFS does not do iterative
 refinement because doing so cannot improve the backward error.\end{DoxyVerb}
 
\end{DoxyParagraph}
