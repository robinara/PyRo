{\bfseries D\+O\+R\+M2\+R} multiplies a general matrix by the orthogonal matrix from a Q\+R factorization determined by sgeqrf (unblocked algorithm). 

 \begin{DoxyParagraph}{Purpose\+: }
\begin{DoxyVerb} DORM2R overwrites the general real m by n matrix C with

       Q * C  if SIDE = 'L' and TRANS = 'N', or

       Q**T* C  if SIDE = 'L' and TRANS = 'T', or

       C * Q  if SIDE = 'R' and TRANS = 'N', or

       C * Q**T if SIDE = 'R' and TRANS = 'T',

 where Q is a real orthogonal matrix defined as the product of k
 elementary reflectors

       Q = H(1) H(2) . . . H(k)

 as returned by DGEQRF. Q is of order m if SIDE = 'L' and of order n
 if SIDE = 'R'.\end{DoxyVerb}
 
\end{DoxyParagraph}
