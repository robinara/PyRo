{\bfseries Z\+P\+O\+T\+R\+F} V\+A\+R\+I\+A\+N\+T\+: right looking block version of the algorithm, calling Level 3 B\+L\+A\+S. 

{\bfseries Purpose\+:} \begin{DoxyVerb} ZPOTRF computes the Cholesky factorization of a real Hermitian
 positive definite matrix A.

 The factorization has the form
    A = U**H * U,  if UPLO = 'U', or
    A = L  * L**H,  if UPLO = 'L',
 where U is an upper triangular matrix and L is lower triangular.

 This is the right looking block version of the algorithm, calling Level 3 BLAS.\end{DoxyVerb}
