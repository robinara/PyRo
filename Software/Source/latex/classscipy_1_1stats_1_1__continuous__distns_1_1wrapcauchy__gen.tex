\subsection{Detailed Description}
\begin{DoxyVerb}A wrapped Cauchy continuous random variable.

%(before_notes)s

Notes
-----
The probability density function for `wrapcauchy` is::

    wrapcauchy.pdf(x, c) = (1-c**2) / (2*pi*(1+c**2-2*c*cos(x)))

for ``0 <= x <= 2*pi``, ``0 < c < 1``.
