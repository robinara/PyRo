{\bfseries D\+G\+S\+V\+J0} pre-\/processor for the routine sgesvj. 

 \begin{DoxyParagraph}{Purpose\+: }
\begin{DoxyVerb} DGSVJ0 is called from DGESVJ as a pre-processor and that is its main
 purpose. It applies Jacobi rotations in the same way as DGESVJ does, but
 it does not check convergence (stopping criterion). Few tuning
 parameters (marked by [TP]) are available for the implementer.\end{DoxyVerb}
 
\end{DoxyParagraph}
