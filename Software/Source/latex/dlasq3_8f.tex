{\bfseries D\+L\+A\+S\+Q3} checks for deflation, computes a shift and calls dqds. Used by sbdsqr. 

 \begin{DoxyParagraph}{Purpose\+: }
\begin{DoxyVerb} DLASQ3 checks for deflation, computes a shift (TAU) and calls dqds.
 In case of failure it changes shifts, and tries again until output
 is positive.\end{DoxyVerb}
 
\end{DoxyParagraph}
