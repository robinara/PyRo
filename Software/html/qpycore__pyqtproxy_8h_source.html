52 // This class is used as a signal on behalf of Python signals and as a slot on
53 // behalf of Python callables. It is derived from QObject but is not run
54 // through moc. Instead the normal moc-generated methods are handwritten in
55 // order to implement a universal signal or slot. This requires some knowledge
56 // of the internal implementation of signals and slots but it is likely that
57 // they will only change between major Qt versions.
