167  // assertUnitStride() is provided as an optimizing trick. When
168  // vectors are constructed outside the function scope, the optimizer
169  // is unaware that they have unit stride. This function sets the
170  // stride to 1 in the local scope so the optimizer can do copy
171  // propagation & dead code elimination. Obviously, you don't
172  // want to use this routine unless you are certain that the
173  // vectors have unit stride.
